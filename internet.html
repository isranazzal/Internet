Clients and Servers

Computers connected to the Web are called clients and servers.

  • Clients are devices that access the web
  • Servers store web pages, sites, or apps

So What Actually Happens?

  1. Browser goes to DNS server
  2. Sends HTTP request
  3. Receives “200 OK”
  4. Browser displays the page
